### Archivo 'IC_Util.R' ###
Contiene la lógica de la funciones para obtener el contraste de hipótesis de dos poblaciones
-----------------------------------------------------------------------------------------------------------------
Función 1: 'contraste_hipotesis_z_2pob'
Ejecuta la prueba Z para la diferencia de medias de dos poblaciones con varianzas conocidas.

Parámetros de entrada:
- Muestrales: Tamaños de muestra, medias muestrales y varianzas poblacionales.
- Significancia: El valor alpha para determinar las regiones de rechazo.
- Tipo de contraste: Define la hipótesis alternativa conforme a las reglas estadísticas:
    - bilateral (igualdad a cero): Su hipótesis alternativa es que la diferencia sea diferente de cero.
	- mayor_igual: Su hipótesis alternativa es que la diferencia sea menor a cero.
	- menor_igual: Su hipótesis alternativa es que la diferencia sea mayor a cero.

Proceso interno:
1. Cálculo de estadísticos: Obtiene la diferencia de medias y el error estándar de la diferencia.
2. Cálculo de Z: Determina el valor del estadístico de prueba.
3. Determinación de valores críticos: Utiliza la función `qnorm()` de R para encontrar los umbrales de la distribución normal según el alpha y el tipo de contraste.
4. Decisión: Compara el valor calculado contra el crítico y genera el p-valor para determinar si se rechaza la hipótesis nula.
-----------------------------------------------------------------------------------------------------------------
Función 2: 'contraste_proporciones_z_2pob'
Ejecuta la prueba Z para la diferencia de proporciones poblacionales.

Parámetros de entrada:
- Muestrales: Tamaños de muestra ($n_1, n_2$) y número de éxitos ($x_1, x_2$).
- Significancia: El valor alpha para el nivel de confianza.
- Tipo de contraste: Define la hipótesis alternativa para la comparación de proporciones.

Proceso interno:
1. Cálculo de proporciones muestrales y estimación de la proporción mancomunada (pooled).
2. Cálculo del estadístico Z de prueba para proporciones.
3. Determinación de valores críticos utilizando la función qnorm() de R.
4. Decisión: Evaluación del estadístico frente al valor crítico para determinar el rechazo de la hipótesis nula.
-----------------------------------------------------------------------------------------------------------------
Función 3: 'contraste_varianzas_f_2pob'
Ejecuta la prueba F para la razón de varianzas de dos poblaciones.

Parámetros de entrada:
- Muestrales: Tamaños de muestra ($n_1, n_2$) y varianzas muestrales observadas ($s_1^2, s_2^2$).
- Significancia: El valor alpha para determinar el límite de rechazo.
- Tipo de contraste: Define la hipótesis alternativa sobre la relación de variabilidad entre poblaciones.

Proceso interno:
1. Cálculo del estadístico F mediante la razón de varianzas muestrales.
2. Definición de grados de libertad ($n - 1$) para el numerador y denominador.
3. Determinación de valores críticos utilizando la función qf() de R.
4. Decisión: Comparación del valor calculado contra el crítico de la distribución F para concluir sobre la hipótesis.
-----------------------------------------------------------------------------------------------------------------
### Archivo 'CasosDeuso.R' ###
Este script actúa como un banco de pruebas para ejercicios generales:
- Ejercicio 15: Evalúa la igualdad de medias entre dos tipos de lectores láser mediante un contraste bilateral con alpha 0.01.
- Ejercicio 16: Realiza un contraste de tipo menor o igual para comparar la calidad de proveedores de circuitos, procesando primero los vectores de datos para obtener sus medias.
- Ejercicio 18: Evalúa la proporción de créditos incobrables entre diferentes estados civiles mediante una prueba de proporciones con alpha 0.01.
- Ejercicio 19: Analiza la variabilidad en las calificaciones de exámenes entre instructores mediante una prueba F con alpha 0.10.
- Ejercicio 20: Compara las varianzas del número de lecturas por segundo entre lectores láser nuevos y antiguos con alpha 0.01.

### Archivo 'Ejercicio17.R' ###
Script dedicado exclusivamente a la resolución del problema de contenido de sodio en bebidas, en donde se aplica un contraste de tipo mayor o igual para verificar si el contenido de sodio en el sabor ginger ale es menor al de cola.
